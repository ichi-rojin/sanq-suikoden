## 0. パック執筆の憲法 ― 初期条件と運命の分離

世界パックは「北宋末という舞台」を記述する。「水滸伝という筋書き」を記述してはならない。
本書全体を貫く規律：

1. **開幕時点主義**：初期データは物語開始時点（宣和年間初頭）の状態のみを記述する。林冲は禁軍槍棒教頭*在職*であり、彼が流罪になるかは世界が決める。
2. **運命の白紙原則**：原典の展開を誘導する初期関係・初期怨恨・特殊フラグを一切仕込まない。原典で*物語開始前から*存在した関係（血縁・婚姻・師弟・同僚）のみ初期関係として登録可。
   - 可：武松—武大（兄弟）、宋江—宋清（兄弟）、孫立—孫新（兄弟）
   - 不可：林冲→高俅の怨恨（原典でも開幕後の事件が原因）、魯智深—林冲の義兄弟（開幕後に成立）
3. **偏在の禁止**：梁山泊を特別扱いしない。水泊という地形的に優れた山寨適地の一つに過ぎず、二龍山や清風山が時代の中心になる世界線は正当な歴史である。
4. **検証**：本パックで10シード回したとき、主要人物の人生が**シードごとに分岐する**こと（§13の運命多様性試験）。全シードで林冲が入山するなら、どこかに台本が漏れている。

## 1. 地理定義

### 1.1 ノード型と属性

エンジンの地理スキーマに与える型（terrain種は既存プリミティブの範囲）：

| ノード型 | 属性の傾向 | 遭遇補正 |
|---|---|---|
| 府城（大都市） | 富・人口・執行力すべて高 | 官吏・禁軍と遭遇 |
| 県城 | 中規模。執行力は腐敗度に脆弱 | 事件の主舞台 |
| 村鎮 | 富低・執行力低 | 盗賊被害・義侠の見せ場 |
| 山寨適地 | 非都市。防御地形値が高い | 放浪者・盗賊集積 |
| 水泊 | 山寨適地の亜種。水戦地形 | 舟遭遇・官軍の攻撃困難 |
| 渡し場・宿場・酒店 | 通過必須点 | **遭遇率ボーナス最大** |
| 関所 | 街道上の制度執行点 | 検問・護送イベント |

各ノード共通属性：富／人口／民心／治安／官府執行力／防御地形値。

### 1.2 地域構成（約70ノード、5地方）

データ入力はM3の作業とし、本書は骨格と代表ノードを確定する。

- **京畿**（開封=東京、中枢。朝廷所在。富と腐敗の頂点）
- **河北・山東**（大名府、鄆城県、陽谷県、清河県、済州、青州、登州…水泊梁山、二龍山、桃花山、清風山。**本パックの主戦場。県城と山寨適地が濃く混在**）
- **河南・関西**（渭州、延安府、華州、孟州、滄州。辺境軍務と流刑地）
- **江南**（江州、揚州、蘇州、杭州。富裕・重税・花石綱の被害地。蜂起構造の温床）
- **辺境北**（雁門、薊州。外圧の入口）

設計意図：山東に県城（執行力が中途半端＝冤罪が起きるが逃げられる）と山寨適地を隣接させ、**追放構造→放浪→入山の地理的動線を短くする**。江南は執行力が高く富も高いため、没落は起きにくいが起きれば蜂起（3-9）へ直結する——地方ごとに発火しやすい構造が異なる地理を作る。

### 1.3 街道と移動

主要幹線（開封—大名府—山東線、開封—江南運河線）＋支線。護送は幹線を通る規則（制度§3.3）のため、**幹線沿いの森・渡し場が救出劇（3-7）の名所**として自然に機能する。

## 3. 制度定義群 ― 本パックの心臓

構造設計書§2.3の制度スキーマ `{管轄, 規則群, ポスト, 腐敗度, 正統性}` に与える実体。
水滸伝の主旋律「体制が好漢を野に追う」は、以下の制度パラメータの合成として鳴る。

### 3.1 朝廷
- 管轄：全土（名目）。正統性：高（ただし減衰イベントあり）。腐敗度：**高（0.7帯）が初期値**。
- ポスト：太尉・知府・知県・都監・提轄・教頭・押司…（ポストは行動候補と俸給と面子を与える）。
- 登用規則：科挙・武挙（腐敗度が高いほど賄賂経路が実力経路を上回る→**有能で貧しい者が制度に入れない**勾配の恒常供給）。

### 3.2 刑罰制度（追放構造の駆動器）
- 冤罪率 = f(所轄の腐敗度, 讒言の有無, 被告の後ろ盾)。
- 刑：笞・徒・流（刺配）・死。**刺配＝顔への入墨＋辺地流刑**。
- **烙印機構**：刺配を受けた者は恒久的な社会属性「罪人の烙印」を得る。烙印は官途の効用を大幅に下げ、江湖での受容度を上げる。→ **エンジン変更要求 #1**（§11）：恒久的社会烙印属性の追加。原典で好漢の多くが「金印を頬に持つ」ことは装飾ではなく、**体制に戻れない構造**そのものである。
- 護送規則：流刑者は幹線経由で護送される（護送ポスト＝防送公人）。護送は襲撃・救出・買収イベントの舞台装置。

### 3.3 招安制度（取込構造の実体）
- 発動条件は制度側の効用（討伐コスト＞叙勲コスト）。受諾側は勢力内の価値観分布で割れる。
- 招安後の配属規則：**辺境の対外戦・対蜂起戦へ優先投入**（腐敗した朝廷の使い潰し）。取込→消耗→残党→再放浪、の悲劇循環は制度パラメータの帰結であり、台本ではない。

### 3.4 賦役・花石綱
- 江南ノード群に対する徴発規則。民心を継続的に削る。**蜂起構造（3-9）の主燃料**。

### 3.5 保甲・民兵
- 県城・村鎮の治安執行。執行力が低いと山寨が育ち、高すぎると民心を削る——治安と自由のトレードオフを地方差として持つ。

### 3.6 禁軍・辺軍
- 軍ポストの供給源。辺軍は外圧攪拌（§7）と連動して需要が変動し、没落軍人（楊志型）や招安の受け皿になる。

| 評価場面 | 上書き | 意図（任侠の美学） |

| 不義の目撃 | 利他反応の重み ×2.0 | 見知らぬ他人のために体制と敵対する非合理を可能にする |

**酒**：飲酒を一時状態バフとして扱う（エンジン変更要求 #2、軽微：一時的評価修飾の仕組み。想起トリガー（宴会の口論）と同一基盤で実装可能なはず）。

## 8. 語彙パック

- **人名生成**：北宋姓プール（張王李趙劉…）＋名の型（単字/双字、輩行字）。
- **渾名生成**：原典の渾名を型分類し生成規則化——
  ①獣＋身体（豹子頭・青面獣）②天象霊獣（入雲龍・混江龍）③古人比擬（小李広・病関索）④職能誇張（神行太保・神医）⑤性向（急先鋒・拚命三郎）。素質・逸話（初勝利・奇行イベント）をキーに付与。**渾名は生まれつきではなく事件で獲得する**（ulo-aquarium称号再評価の機構を流用）。
- **題テンプレ**：基本形「{地名}の{戦い|乱|変}」「{渾名}{救出|反乱|最期}」。上級形として**対句形式**（「林教頭風雪山神廟」型の七字対句）をLLM装飾層の目標文体に指定（テンプレ層では作らない——G5/G15合意事項）。
- **台詞語彙**：哥哥/兄弟/好漢/義に背くか/一献酌み交わそう…場面×性格×関係キーの台詞テーブル。演義調の史官文体定義。

## 10. 名声の二重化 ― 官途名声と江湖名声

パック設計中に判明した本質：水滸伝の人物は**二つの世間**を生きる。官界での地位と、江湖（アウトロー社会）での評判は別勘定であり、しばしば逆相関する（宋江は押司としては小吏だが江湖では及時雨）。
- **エンジン変更要求 #3**：名声を単一値ではなく**観衆（コミュニティ）別のスコープ付き名声**にする。エンジンは「名声はコミュニティごと」という汎用機構を持ち、パックが観衆（官界/江湖/民間）を定義する。
- これが無いと、招安の魅力（江湖名声を官途名声へ換金する制度）と、その悲劇（換金したら江湖の信を失う）が表現できない。**取込構造の心理的核**であるため、Phase1必須と判断する。

## 13. 検証 ― 運命多様性試験

憲法§0の担保。10シード×100年で計測：
1. S層12名それぞれの人生終着（官途全う/入山/独立/横死/隠棲…）が**シード間で分岐する**こと。特定人物の終着が10/10一致したら台本混入を疑い監査。
2. 「梁山泊が最大山寨になる」シード比率が過半を大きく超えないこと（偏在の禁止の検証）。
3. 一方で、らしさメトリクス（§12）は全シードで帯内であること。**個々の運命は自由、世界の基調は不変**——これが「歴史が生まれる」の操作的定義である。

## 14. 批判的レビュー（G17）

**14-1. 価値観翻訳は解釈の押し付けではないか**：林冲=忠誠85は一つの読解に過ぎない。→ 認める。S層の数値には出典逸話の注記を付け、解釈の変更をデータ改訂として扱える形にする。エンジンは無傷。

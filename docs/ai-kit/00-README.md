# AI Development Kit v1

本キットは「水滸世界シミュレーション／歴史生成エンジン」プロジェクトの設計品質を、
特定AI（Fable 5）への依存から**設計組織への依存**へ移行するための運用体系である。
ただし各文書はプロジェクト固有部と汎用部を分離してあり、**他プロジェクトへ再利用可能**。

## 構成（優先順位順）

| # | 文書 | 用途 |
|---|---|---|
| 01 | master-prompt.md | 主任設計者の頭脳の再現。高性能モデルのセッション冒頭に貼る |
| 02 | orchestration-manual.md | AIチーム運営マニュアル（責務・受け渡し・台帳・復旧） |
| 03 | expert-roster.md | 専門家ロスター最終版（帽子一覧） |
| 04 | prompt-library.md | AI別・用途別の完成プロンプト集 |
| 05 | dev-flowchart.md | 完成までの運用フロー（条件分岐込み） |
| 06 | tacit-knowledge.md | 暗黙知の形式知化（判断基準の全記録） |

## 使い方

1. **新セッション開始時**：01を貼る → 02のセッション開始テンプレートに従う → 必要な設計書（下記）を添付。
2. **作業種別が決まっている時**：04から該当プロンプトをコピーし、添付欄を埋めて貼る。
3. **迷った時**：05のフローチャートで現在地を確認 → 06で判断基準を参照。

## 最上位規範（全文書に優先する五憲法＋二重検問）

憲法：①因果保存（全イベントにcauses[]）②観測不干渉 ③環境介入限定 ④固有名詞境界（if文はエンジン、名詞はパック）⑤決定論（シード＋ログで完全リプレイ）
二重検問：「プレイヤーを削除しても面白いか」「再現か、生成か」
**いかなるAIのいかなる裁定も、この五憲法と二重検問に劣後する。**
